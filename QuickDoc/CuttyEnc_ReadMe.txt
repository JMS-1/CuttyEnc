CuttyEnc - Encoding Provider to use MPEG2Enc with Cuttermaran

CuttyEnc allows Cuttermaran to use MPEG2Enc.EXE to re-encode MPEG-2
streams on cut points. Re-encoding is necessary if the a cut sequence
does not start on an I-frame (full picture) or ends on a B-frame
(bidirectional difference picture). Like Cuttermaran MPEG2Enc is free under
the GPL. The quality of the encoded material is only a bit worse compared 
to the TMPGEnc 2.51 result but in many real scenarios no difference can be
detected when viewing the resulting video.

To get the latest version of CuttyEnc please visit the official homepage
http://www.psimarron.net/Projects/DVD/CuttyEnc.

MPEG2Enc.exe is part of the DVDAuthor package, which can be obtained
from http://dvdauthor.sourceforge.net/.

INSTALLATION
- Unpack CuttyEnc.zip to Cuttermaran folder Provider\Encode (this will 
  add 3 files to the Encode folder and some language DLL's to the 
  corresponding language folders).
- Unpack the DVDAuthor package for example to the Cuttermaran folder
  External Tools (it may be necessary to create this folder).
- Open Cuttermaran and use the Actions/Settings... menu to open the 
  configuration dialog. On the Encoding tab select MPEG2Enc as
  encoder and use the Configure... button to select the folder containing 
  MPEG2Enc.EXE.

CuttyEnc - Cuttermaran Encoding Schnittstelle für MPEG2Enc

CuttyEnc ermöglich Cuttermaran die Nutzung von MPEG2Enc zum Re-Encode
von MPEG-2 Material an Schnittpunkten. Dieses ist notwendig, wenn
der Schnitt nicht an einem sogenannten I-Frame (Vollbild) beginnt oder
auf einem B-Frame (doppeltes Differenzbild) endet. Genau wie Cuttermaran
ist MPEG2Enc Freeware unter der GPL. Die Qualität des Re-Encode ist
ein wenig schlechter als die des kommerziellen Produktes TMPGEnc 2.51, 
jedoch ist dieser Unterschied in vielen Fällen beim Betrachten des 
geschnittenen Films nicht feststellbar.

Um eine aktuelle Version von CuttyEnc zu erhalten besuchen Sie bitte
die offizielle Homepage http://www.psimarron.net/Projects/DVD/CuttyEnc.

MPEG2Enc.exe ist im DVDAuthor Paket enthalten, das über 
http://dvdauthor.sourceforge.net/ verfügbar ist.

INSTALLATION
- Entpacken Sie CuttyEnc.zip in den Provider\Encode Ordner von Cuttermaran
  (in diesen werden 3 Dateien abgelegt sowie weitere Sprachdateien in die
  ensprechenden Unterverzeichnisse).
- Entpacken Sie das DVDAuthor Paket zum Beispiel in den External Tools
  Ordner von Cuttermaran (der Ordner muss eventuell angelegt werden).
- Öffnen Sie Cuttermaran und wählen Sie über das Menü Aktionen/Einstellungen...
  den Konfigurationsdialog und dort die Registerkarte Encoding. Zur 
  Aktivierung von CuttyEnc wird als Encoder MPEG2Enc ausgewählt. Nach
  Betätigen der Schaltfläche Einstellungen... wählen Sie dann noch den
  Ordner aus, in dem sich MPEG2Enc.EXE befindet.

CuttyEnc - Aanstuurprogramma voor MPEG2Enc in Cuttermaran

CuttyEnc maakt het mogelijk dat Cuttermaran gebruik kan maken van  MPEG2Enc
om MPEG2 bestanden te hercoderen op knip punten. Dit hercoderen is nodig 
als het knip moment niet begint op een I-frame (= volledig frame) of eindigt
op een B-frame ('bidirectioneel' verschil frame). 
Zoals Cuttermaran, is ook MPEG2Enc vrij beschikbaar onder de GPL. 
Bij gedetailleerde analyse is de kwaliteit van het ge-hercodeerd materiaal
slechts een beetje slechter dan bij het gebruik van het commerciele TMPGEnc. 
Echter, onder praktische condities is het verschil nauwelijks zichtbaar. 

De meest recente versie van CuttyEnc kan gevonden worden of de officiele
homepage http://www.psimarron.net/Projects/DVD/CuttyEnc.
MPEG2Enc.exe is onderdeel van het DVDAuthor-pakket, wat verkregen kan worden
via http://dvdauthor.sourceforge.net/.

INSTALLATIE
- Pak CuttyEnc.zip uit in Cuttermaran's map Provider\Encode (dit zal 3 
  bestanden toevoegen aan de Encode-map en wat taal dll's aan de taal 
  mappen).
- Pak het DVDAuthor-pakket uit in b.v. een map Cuttermaran/External Tools
  (moet eventueel eerst nog gemaakt worden). 
- Open Cuttermaran, via Aktie/Instellingen...Encoder, selecteer MPEG2Enc
  als encoder en selecteer de map met MPEG2Enc.exe (via Instellen).
